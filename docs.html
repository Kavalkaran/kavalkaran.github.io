Introduction

Kavalkaran is a public-interest cybersecurity tools and awareness platform designed to assist citizens in understanding and responding to common cyber threats. This documentation provides general operational guidance for users.

Platform Access

You can access Kavalkaran through web browsers on desktop and mobile devices. No special software installation is required. Ensure your browser is up to date for the best experience.

How to use tools

Kavalkaran offers various cybersecurity tools such as link scanners, file lockers, QR code Saftey Checker. Navigate to the respective tool sections on the platform to utilize these features.

  • Ensure information accuracy
  • Avoid submission of false data
  • Do not include unnecessary personal information

Dashboard Usage

The dashboard provides summarized views of alerts, awareness materials, and system notifications. Displayed information may be delayed or incomplete depending on system conditions.

Data Handling

Data processed within the platform is handled in accordance with applicable policies and technical safeguards. No absolute guarantee is made regarding data retention or persistence.

Limitations

Kavalkaran does not guarantee prevention, investigation, or resolution of cyber incidents. The platform is informational in nature and does not replace official law enforcement or professional services.

Privacy Compliance

User data may be processed in accordance with published privacy documentation. Continued use of the platform implies acceptance of such policies.
